变量命名，在适当的时候添加统一的前缀或者后缀

camelCase

- js 项目文件命名
- js 项目变量命名

PacalCase

- js 项目 Type 命名
- js 项目组件命名

snake_case

- python/c/c++/rust/bash/自动化流水线等项目配置文件
- python/c/c++/rust/bash 项目文件命名
- python/c/c++/rust/bash 项目变量命名

SCREAMING_SNAKE_CASE

- 常量

kebab-case

- js 项目命名
- js 项目文件命名
- js 脚本命名
- js 项目的配置文件命名
- js 项目插件命名
