# 2. Svn

## 1. Requirement

The following stuff is needed for the installation.

Apr
{% highlight html %}
$ ./configure --prefix=/usr/local/apr
$ make
$ sudo make install
{% endhighlight %}

Apr-util (must have prefix in order to be used in installation of Subversion)
{% highlight html %}
$ ./configure --prefix=/usr/local/apr-util
$ make
$ sudo make install
{% endhighlight %}

SQLLite
{% highlight html %}
$ sudo apt-get install sqlite3 libsqlite3-dev
{% endhighlight %}

Zlib
{% highlight html %}
$ sudo apt-get install zlib1g zlib1g-dev
{% endhighlight %}


## 2. Install Subversion

Download form official site and install.
{% highlight html %}
$ ./configure --with-apr=/usr/local/apr/ --with-apr-util=/usr/local/apr-util/
$ make
$ sudo make install
{% endhighlight %}

## 3. Create Local SVN Repository

{% highlight html %}
// Checkout
$ svn checkout http://example.com
{% endhighlight %}

## 4. Submit Changes

{% highlight html %}
Update from server and then fix the conflicts if exists.
$ svn update

// Add all new files
$ svn add ./*

// Delete files
$ svn delete test.txt

// Commit
$ svn commit -m"add test"
{% endhighlight %}

## 5. Integrate SVN in Intellij
